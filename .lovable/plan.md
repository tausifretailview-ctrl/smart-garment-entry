

# Windows Native UI - Ezzy ERP Theme

## Completed: WinUI/Fluent Design Implementation

### Color System
- **Background**: #F5F7FA (Light Windows gray)
- **Cards**: Pure white with subtle shadow
- **Primary Accent**: #2563EB (Windows Blue)
- **Sidebar**: #0F172A (Dark Slate)
- **Text**: Dark gray hierarchy with muted labels

### Dashboard Cards
- White background with 1px border
- Colored left border accent (no gradient fills)
- Large dark value text (text-2xl font-semibold)
- Muted uppercase label text (text-xs)
- Subtle hover elevation effect

### Buttons (Windows-style)
- Primary: Blue solid (#2563EB) with white text
- Secondary: White with gray border
- Rounded 6px corners
- Height: h-9 (default), h-8 (sm), h-10 (lg)
- Clean hover/active states (no scale transforms)

### Charts
- Single primary color scheme (#2563EB blue)
- Clean minimal design
- No gradients
- White card containers

### Typography
- font-medium/font-semibold (not bold)
- Clear hierarchy: titles, values, labels
- Muted foreground for secondary text

### Overall Feel
- Enterprise calm aesthetic
- Windows native appearance
- SaaS professional (Zoho/Vyapar style)
- Optimized for PWA desktop app
